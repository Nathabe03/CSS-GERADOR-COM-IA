⚡ Gerador de Código com IA
Este é um projeto que utiliza Inteligência Artificial para transformar descrições em texto em códigos reais de HTML e CSS. Você digita o que imagina, e a IA cria o design e a animação para você na hora.

🛠️ O que o projeto faz?
Entende seu pedido: Você escreve "Crie um botão que brilha" ou "Uma bola verde quicando".

Gera o código: O sistema se conecta com a API da Groq (Llama 3) e recebe o código pronto.

Mostra o resultado: O site divide a tela em duas partes: de um lado você vê o texto do código e do outro você vê o desenho funcionando (Preview).

🚀 Tecnologias utilizadas
HTML5: Para a estrutura das caixas e botões.

CSS3: Para deixar o visual moderno, escuro e organizado com Flexbox.

JavaScript: Para fazer a "ponte" entre o seu site e a Inteligência Artificial usando fetch e async/await.

⚠️ Observação Importante: Segurança e API Key
Por questões de segurança e boas práticas de desenvolvimento, a chave de API original foi removida do código público.

Para testar as funcionalidades de geração de código, você precisará utilizar a sua própria chave da Groq. Isso garante que seus créditos e limites sejam geridos de forma privada.

🚀 Como testar o projeto
Obtenha sua chave: Crie uma conta gratuita em Groq Cloud e gere uma API Key.

Clone o projeto: Baixe os arquivos para o seu computador.

Abra o site: Execute o arquivo index.html no seu navegador.

Configure a chave: No campo indicado "API KEY", cole a sua chave gsk_....

Gere seu código: Descreva o elemento desejado e clique em Gerar Código ⚡.
